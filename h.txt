Feature: Inicio de sesión en SwagLabs
  Como usuario de SwagLabs
  Quiero poder iniciar sesión en la plataforma
  Para acceder a mis productos y realizar compras

  Background:
    Given que el usuario se encuentra en la página de inicio de SwagLabs

  @login-success
  Scenario: Inicio de sesión exitoso con credenciales válidas
    When el usuario ingresa el usuario "standard_user" y la contraseña "secret_sauce"
    And hace clic en el botón de inicio de sesión
    Then el sistema muestra la página principal con productos disponibles

  @login-error
  Scenario: Inicio de sesión fallido con contraseña incorrecta
    When el usuario ingresa el usuario "standard_user" y la contraseña "incorrect_password"
    And hace clic en el botón de inicio de sesión
    Then el sistema muestra un mensaje de error que indica credenciales incorrectas

  @login-locked
  Scenario: Inicio de sesión con un usuario bloqueado
    When el usuario ingresa el usuario "locked_out_user" y la contraseña "secret_sauce"
    And hace clic en el botón de inicio de sesión
    Then el sistema muestra un mensaje que indica que la cuenta está bloqueada

  @login-missing
  Scenario: Inicio de sesión con campos vacíos
    When el usuario deja vacío el campo de usuario y contraseña
    And hace clic en el botón de inicio de sesión
    Then el sistema muestra un mensaje que indica que los campos son obligatorios
